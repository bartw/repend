# Hello

Let's get this project started.
Whip up a new shell.

```shell
mkdir repend
cd repend
git init
npm init
touch index.js
```

Then put the following code in the index.js file.

```js
(function () {
    console.log('Hello!');
})();
```

Back to the shell.

```shell
npm start
```

Step 1 finished, the hard part is over.
